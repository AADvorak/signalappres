App description

This web app models signal transformations
App abilities:
  • signal generation
  • signal transformation
  • signal store in database
  • signal export / import from files (in development)
  • extending functionality by additional modules

Signal generation

Use signal generator to generate signals and import from files

Signal management

Use signal manager to see / edit / delete your stored signal, send them to transformers, view, export to files, add several signals.
Special module cable helps to process signals. Every time you have imported / generated new signal or done some transformation, the result is opened with cable, when you can preview it, edit description. Then you can save it or do further transformations.

Modules management

Write your own plugins to extend app functionality. Modules manager helps you to register your modules and tell app how to deal with them.
